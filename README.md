# Домашняя работа № 3. Python для ETL (итоговый проект)

## Системные требования
* Python >= 3.10
* Docker-compose >= 1.29.2

## Установка
* docker-compose up 
* pip install pipenv
* pipenv install `(пакеты для запуска)`
* pipenv install --dev `(пакеты для разработки)`

## Запуск
* pipenv shell
* python main.py

## Проверка оформления кода
* flake8
